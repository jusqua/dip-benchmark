\section{Metodologia}

\subsection{Desenvolvimento dos operadores}

Nesta etapa, escolhemos algoritmos de processamento de imagens existentes na VisionGL para adaptação dos conceitos originais para as ferramentas que buscamos analizar algumas operações ponto a ponto e janela.

Além disso, é necessário analisar a transferência de dados entre a memória do dispositivo e a memória do hospedeiro, bem como o gerenciamento de memória para garantir que os dados sejam acessados de forma eficiente durante a execução dos operadores.
