\section{Metodologia}

A metodologia adotada será baseada nos experimentos de \cite{Dantas2015opencl} que utiliza a biblioteca VisionGL com operadores de processamento de imagens implementados em OpenCL como base para análise de desempenho de diferentes tecnologias de computação paralela. Repetiremos o experimento com as mesmas e novas bibliotecas e tecnologias:
\begin{itemize}
    \item SYCL através do compilador AdaptiveCPP.
	\item CUDA com Julia através da biblioteca CUDA.jl.
	\item OpenCV com Python através da biblioteca opencv-python.
	\item OpenCL através da biblioteca VisionGL.
	\item MATLAB Parallel Computing Toolbox.
\end{itemize}

A foi feito um \textit{fork} da biblioteca VisionGL\footnote{\url{https://github.com/ddantas/visiongl}} para estudo da estrutura do projeto e desenvolvimento dos operadores lógicos em outras tecnologia. O \textit{fork} este presente no repositório do GitHub\footnote{\url{https://github.com/jusqua/visiongl}}.

Desenvolveremos os operadores de processamento de imagens descritos conforme as \Crefrange{eq:grayscale}{eq:dilation} nas ferramentas que buscamos analizar o desempenho. Além disso, é necessário também analisar a transferência de dados entre a memória do dispositivo e a memória do hospedeiro, bem como o gerenciamento de memória para garantir que os dados sejam acessados de forma eficiente durante a execução dos operadores.
